# air-gapped-toolkit

## 왜?

폐쇄망 환경(ex - 증권사에서 제공하는 DMA) 내에서 개발을 해야 한느 경우  
순수한 vim과 gcc만으로 개발을 해야 하는 상황이 있음.  
개인적으로 쓰는 .vimrc나 bash, 직접 만든 vim plugin 등을 저장해두자.
